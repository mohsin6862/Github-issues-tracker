
## Q1 What is the difference between var, let, and const?
## Answer:
 - Var is old way of declaring variables 
 - let can not be redeclared but Can be reassigned
 - const can not be redeclared but Can not be reassigned

## Answer:

## Q3  What is the difference between map(), filter(), and forEach()?
## Answer:
 - map() returns a new array which has same length as original array
 - filter() is used when we want to find spaceific things based on a condition.
 - forEach() is used when we need  every element, but it does not return a new array.
